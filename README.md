### プロジェクト
#### プロダクト名
##### campus de lunch

#### 概要
* 大学でご飯を食べようと思った際に，友人が構内にいるかを確認しご飯に誘うことができるサービス
* 友人は自分で構内にいるかどうかのレスポンスをする必要がなく，アプリがバックグラウンドで'特定の場所にいるかいないか'の情報のみを返す．
